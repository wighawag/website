Profiler Injection With MixingLoom as3commons bytecode and Flash Preloader (draft)

Sunday, June 5, 2011
Introduction Recently I discovered MixingLoom through James Ward’s article, it is a library to ease the injection of code in swf. One usage of it is Aspect Oriented Programming to separate logging, analytics… from the actual application code by injecting the extra code (logging, analytics…) into the swf after compilation (thanks to as3commons-bytecode library ). This way the extra code does not appear anywhere in the application source code which stay focused on what it should do.
